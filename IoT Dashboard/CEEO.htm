Hey kids! So you've been browsing different webpages and powerpoints and found some cool widgets that you wanna play around with, huh? Well, this webpage lets you rearrange the widgets to your liking and help you learn some programming along the way!

To proceed, we need to make sure that we really understand what exactly a widget is. Widgets are extensions of other applications or websites that are connected to data streams, which allow the widget to display information in real time. The most common types of widgets are those on our phones, such as the time, weather, and news widgets. These mini boxes can be rearranged according to will and show live updates.

Of the many widgets you have seen, let's say that you liked the ones below. These widgets will be in your library and can be accessed by a drop-down menu. You can drag and drop them onto the dashboard to explore them, and then rearrange these widgets in any order that you like.

A simple way to understand this would be using Excel graphs. Let's say that we have a set of data on one sheet, with a graph for that data on another sheet. If the data is changed, the graph will change to reflect this change immediately. This graph can be thought of as a widget too. Now make a bunch of such data sheets and graphs, and we have a dashboard of graph widgets!
